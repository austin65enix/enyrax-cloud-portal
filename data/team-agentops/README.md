# Team_AgentOps Demo Data Fixtures

These files are safe demo fixtures for Team_AgentOps.

They are designed to match the schema concept from `docs/team_agentops_agent_run_log_schema_design.md`.

They do not contain real prompts, responses, raw sessions, command output, file diffs, credentials, secrets, API keys, tokens, private keys, or full home paths.

Token / cost values are operational demo estimates, not billing-grade data.

Team scorecard values are governance demo metrics, not employee surveillance scores.
